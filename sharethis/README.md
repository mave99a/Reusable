# ShareThis Placeholder App

## Overview
This package reserves a namespace for integrating the ShareThis widget within a
Django project. It currently contains no code but can be used as a convenient
hook for static assets or template tags related to ShareThis integration.

## Suggested usage
* Add ShareThis JavaScript snippets or helper templates under this package.
* Bundle Django template tags or context processors that generate ShareThis
  metadata.

Feel free to expand the app to match the needs of your project.
